# Blob Demo

This is a small open-source demo that recreates the chrome/snow blob scene from your screenshot using WebGL (Three.js).

## Quick start

1. Install deps:
   `npm install`
2. Run dev server:
   `npm run dev`
3. Open `http://localhost:5173`

## Build

`npm run build`

## Caption

This version renders the scene without caption text.
